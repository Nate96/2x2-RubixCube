BUILD
1. Download Main.py from Canvas
2. Open any application that can interpret Python
3. Run Main.py

GUI
1. Grid Tab: allows you to manipulate the state of the puzzle
	a. Reset: resets the puzzle to the intial (goal) state
	b. Random State: randomizes the tiles in the puzzle
	c. Input State: customize the position of the tiles manually
2. Solve Tab: allows you to solve the unsolved puzzle with one of three search algorithms
	a. Breadth First Search
	b. A* Search
	c. Greedy Best First Search
3. Puzzle: tiles can be moved around by clicking on tiles adjacent to the blank tile
